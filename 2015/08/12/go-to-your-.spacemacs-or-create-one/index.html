Go to your .spacemacs (or create one)

August 12, 2015

From anywhere in Spacemacs, you can run SPC f e d and you’ll get your .spacemacs file.

NB: If you have .spacemacs symlinked to a file in version control (for managing your dotfiles all in one place, for instance) Spacemacs will ask if you want to edit that file instead, but the prompt sometimes get hidden. Hit “y” if it’s hanging.

Update from Swaroop: (setq vc-follow-symlinks t) is one way to avoid the ‘y/n’ prompt if you are using version control.
